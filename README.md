## GitHub图片不显示问题：

Mac终端输入：`sudo vi /etc/hosts`
输入密码`enter`后，点击 `i`键，进入Insert模式,
如果hosts文件已存在，点击`e`进入然后点击 `i`键，进入Insert模式。

将下面内容拷贝进去:
```
# GitHub Start

151.101.184.133 raw.githubusercontent.com

# GitHub End
```
